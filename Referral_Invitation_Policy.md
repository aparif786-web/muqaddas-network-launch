## 2. Eligibility

1. **Current Users**: Only registered users with verified accounts can participate in the referral program.

## 3. Invitation Process

### 3.1 How to Invite Users

Users can invite friends through:
- **Unique Referral Link**: Each user receives a unique link upon registration.
- **Social Media Sharing**: Options to share the referral link via social media platforms.

### 3.2 Invitation Message

Invitations should include:
- A personal message (optional).
- A brief description of the benefits of joining the Muqaddas Network.
- Information about the joining fee (₹1).
